PAI-Bot Installation Tutorial
------------------------------

- prerequisite
    - Node.JS

- github repository
    - download/clone repository
    - https://github.com/PAI-Tech/PAI-BOT

- npm Install
    - install all packages + update

- bot config
    - run npm run config
    - select http connector (port 3141 no ssl)

- run bot
    - run npm run bot + open browser localhost:3141

- special feature:
    - connect bot to pai-net
    - show bot on console
